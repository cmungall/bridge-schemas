    Enum: CogFunctionalCategory

    COG (Clusters of Orthologous Groups) single-letter functional categories. Multi-letter combinations indicate proteins with multiple functions.

    CATEGORY FREQUENCIES IN DATABASE: - S (unknown): Most common - J (translation): ~8% of annotations - K (transcription): ~6% - C (energy): ~5% - E (amino acids): ~5%

    URI: https://w3id.org/kbase/kbase_ke_pangenome/CogFunctionalCategory

    Permissible Values

    Value Meaning Description
    A None RNA processing and modification
    B None Chromatin structure and dynamics
    C None Energy production and conversion
    D None Cell cycle control, cell division, chromosome partitioning
    E None Amino acid transport and metabolism
    F None Nucleotide transport and metabolism
    G None Carbohydrate transport and metabolism
    H None Coenzyme transport and metabolism
    I None Lipid transport and metabolism
    J None Translation, ribosomal structure and biogenesis
    K None Transcription
    L None Replication, recombination and repair
    M None Cell wall/membrane/envelope biogenesis
    N None Cell motility
    O None Posttranslational modification, protein turnover, chaperones
    P None Inorganic ion transport and metabolism
    Q None Secondary metabolites biosynthesis, transport and catabolism
    R None General function prediction only
    S None Function unknown
    T None Signal transduction mechanisms
    U None Intracellular trafficking, secretion, and vesicular transport
    V None Defense mechanisms
    W None Extracellular structures
    X None Mobilome - prophages, transposons
    Y None Nuclear structure
    Z None Cytoskeleton
